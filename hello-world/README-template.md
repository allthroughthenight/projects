# Lang

### Information
* History: Made by Bob to better make templates in 1985. Major use case: READMEs.
* Paradigm(s): Functionally Objective, Strongly Opinionated
* Typing: Overt, Covert

### Set-Up
<pre>
$ sudo apt install -y build lang
</pre>

### Build
<pre>
$ build hello.lang -o hello
</pre>

### Run
<pre>
$ run hello
</pre>
